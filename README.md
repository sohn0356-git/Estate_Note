# Estate Note

Estate Note는 매일 공부하면서 알게 된 내용을 정리하는 개인 웹사이트입니다.

## 프로젝트 구성

- `index.html`: 메인 페이지
- `styles.css`: 스타일
- `script.js`: 간단한 자바스크립트

## 사용 방법

1. `Estate_Note` 디렉토리에서 파일을 수정합니다.
2. 변경 내용을 GitHub에 커밋합니다.
3. GitHub Pages를 사용하면 정적 사이트로 배포할 수 있습니다.

## 앞으로 할 것

- 공부 노트를 Markdown 형식으로 정리하기
- 더 많은 페이지 추가하기
- GitHub Actions로 자동 배포하기
