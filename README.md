# Estate Note

Estate Note는 부동산 공부를 튜토리얼형으로 정리하는 React 기반 웹사이트입니다.

## 프로젝트 구성

- `index.html`: React 루트가 있는 메인 페이지
- `styles.css`: 부동산 앱 영감을 받은 인터페이스 스타일
- `script.js`: React 컴포넌트 및 인터랙티브 기능
- `design.md`: 디자인 참고와 주요 영감 출처

## 기능

- 매일 학습 루틴을 단계별로 정리
- 오늘의 공부 미션 표시
- 실시간 노트 작성 및 검색
- 학습 로드맵과 단계별 코스 구성

## 사용 방법

1. `Estate_Note` 디렉토리에서 파일을 수정합니다.
2. 변경 내용을 GitHub에 커밋합니다.
3. `git push origin main`으로 원격에 반영합니다.

## 앞으로 할 것

- GitHub Pages 배포 자동화
- 공부 기록을 Markdown 페이지로 변환
- 추천 강의 및 기사 링크 추가
- 로그인 기반 개인 목표 관리
